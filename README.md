
# Calculator App

simple Calculator using python-customtkinter module.




## Installation

Clone this repo.

```bash
  git clone https://github.com/adnaaaen/calculator-app.git
```
```bash
  cd calculator-app
```
install required python modules.
```bash
  pip install -r requirement.txt
```
run calculator
```bash
  python3 main.py 
```
